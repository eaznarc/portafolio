Funcion de la API

Esta API funciona al hacer que una funcion se ejecute en el fondo para que la pagina principal no corra todo el codigo al mismo tiempo

Importancia

Permite que las computadoras lentas o con poca capacidad puedan correr la pagina sin que se sature

Por ejemplo, si esta pagina tuviera multiples funciones, aun asi se podria cambiar el color de fondo sin problema ya que el worker se encarga de eso y no la pagina principal
